# 🩺 Diabetes Risk Predictor

A machine learning web application that predicts diabetes risk
based on clinical health parameters.

## 🔗 Live Demo
[Click here to try the app](YOUR_STREAMLIT_URL_HERE)

## 🧠 Model
- Algorithm: Random Forest Classifier
- Dataset: Pima Indians Diabetes Dataset (768 patients)
- Accuracy: ~77%

## 📊 Features Used
- Glucose level, BMI, Blood Pressure, Age, Insulin
- Diabetes Pedigree Function (genetic factor)

## 🛠 Tech Stack
Python · Scikit-learn · Pandas · Streamlit
